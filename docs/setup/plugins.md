# Recommended Plugins

These plugins are tried and tested with Naruto Destination on Crucible servers. All are compatible with Forge 1.7.10 via Crucible's Bukkit support.

---

## Essential Plugins

### Essentials

The go-to plugin for practically everything — spawns, homes, warps, kits, economy, chat formatting, and more.

[:octicons-download-16: Download (v2.14.1.3)](https://dev.bukkit.org/projects/essentials/files/891400){ .md-button }

---

### PermissionsEx

The permission management plugin. Use it to control which commands and features each player group can access. Critical for setting up rank-based permissions with the Naruto Destination command system.

[:octicons-download-16: Download (v1.22.10)](https://github.com/PEXPlugins/PermissionsEx/releases/download/STABLE-1.22.10/PermissionsEx-1.22.10.jar){ .md-button }

!!! tip
    Set up permission groups that mirror your Naruto ranks (Genin, Chunin, Jonin, etc.) for a smooth roleplay experience.

---

### WorldGuard

Protect regions on your map from griefing. Essential for protecting villages, arenas, and important builds.

[:octicons-download-16: Download](https://dev.bukkit.org/projects/worldguard/files/881691){ .md-button }

---

### WorldEdit

Powerful in-game map editing tool. Use it to build your villages, arenas, and landscapes efficiently.

[:octicons-download-16: Download](https://dev.bukkit.org/projects/worldedit/files/880435){ .md-button }

---

## Optional Plugins

### GriefPrevention

Lets your players easily claim and protect their own land using a golden shovel. Great for survival-oriented servers.

[:octicons-download-16: Download](https://dev.bukkit.org/projects/grief-prevention/files/833268){ .md-button }

---

### GreatKits

A fully configurable kit plugin. Set up daily kits for your players — perfect for giving out starter gear, tokens, or consumables on a schedule.

[:octicons-download-16: Download](https://www.spigotmc.org/resources/greatkits-1-7-1-13-full-config-kits-plugin-save-and-load-inventory-with-commands.44535/){ .md-button }

---

### MagicSpells

Allows you to bind commands to items. Useful for creating custom scrolls, special weapons, or clickable tokens that execute mod commands.

[:octicons-download-16: Download](https://dev.bukkit.org/projects/magicspells/files/792963){ .md-button }
