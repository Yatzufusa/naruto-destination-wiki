# Recommended Mods

Companion mods to include in your modpack alongside Naruto Destination.

---

## Server + Client Mods

These mods must be installed on **both** the server and client.

### CustomNPCs+

The enhanced version of CustomNPCs. Used for creating NPCs, quests, traders, and scripted content on your server.

[:octicons-download-16: Download on CurseForge](https://www.curseforge.com/minecraft/mc-mods/customnpc-plus){ .md-button }

!!! tip "Scripting API"
    Check out the [Naruto Destination Scripting API](https://yatzufusa.github.io/naruto-destination-cnpc-scripting-api/) for integrating Naruto Destination with CustomNPC+ scripts.

---

### Barrier Blocks

Adds invisible barrier blocks for map building. Essential for creating arenas, boundaries, and invisible walls.

[:octicons-download-16: Download on CurseForge](https://www.curseforge.com/minecraft/mc-mods/barrier-block-mod){ .md-button }

## Client-Only Mods

These mods only need to be in the client modpack. They enhance the player experience but are not needed on the server.

[:octicons-download-16: Download on CurseForge](https://www.curseforge.com/minecraft/mc-mods/controlling){ .md-button }

## Mods to Avoid

!!! danger "Known Incompatibilities"
    The following mods are known to cause issues with Naruto Destination or Crucible:

    **MrCrayfish's Mods**
    :   These mods have been reported to cause server stability issues. Avoid all mods by MrCrayfish.

    **Forge Multipart**
    :   Not compatible with Crucible. Will cause crashes on startup.

    **FoamFix**
    :   Causes texture bugs with player skins when dojutsus are unlocked. If players report skin glitches after getting a Sharingan or Byakugan, this mod is likely the cause.
