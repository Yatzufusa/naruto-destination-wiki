## What is Naruto Destination?

Naruto Destination is a **multiplayer-focused rework** of the Mathioks Naruto Anime Mod for Minecraft 1.7.10. Key improvements include:

- **Scaling jutsus** — All jutsus scale in damage and cost with your Ninja Info Card stats
- **Mastery system** — Train and improve your favourite jutsus
- **Token economy** — Mobs drop tokens instead of raw XP, enabling proper progression systems
- **Full configurability** — Almost everything is configurable, so every server has its own meta
- **In-game wiki** — A customizable wiki built right into the game
- **Performance** — Significantly improved for both clients and servers

The mod is also playable in singleplayer with Randomizer Traders providing access to all content.

## Addons

Naruto Destination supports addons that extend core functionality:

| Addon | What It Adds |
|-------|-------------|
| **[Villages](villages/index.md)** | Village banks, transport, ranks, territory, PvP rules, diplomatic relations |
| **[Kage Vote](kagevote/index.md)** | Democratic elections for choosing Kages with configurable requirements |
| **[XP Events](xpevents/index.md)** | Manual and automatic XP multiplier events |

Addons require the base Naruto Destination mod (`required-after:NarutoDestination`).
