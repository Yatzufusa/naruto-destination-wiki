# Villages Addon

The **NarutoDestinationVillages** addon adds a complete village management system including banks, transport networks, rank hierarchies, territory control, PvP rules, and diplomatic relations.

---

## Command Overview

### Player Commands

| Command | Aliases | Description |
|---------|---------|-------------|
| [`/villages`](player-commands.md#villages) | `/v`, `/villageshub` | Main village hub &mdash; GUI, stats, bank, transport, ranks, relations |
| [`/villagepvp`](player-commands.md#villagepvp) | `/vpvp` | Check PvP status, rules, and loot information |
| [`/villagerelation`](player-commands.md#villagerelation) | `/vr`, `/relation` | View and manage diplomatic relations |

### Leadership Commands

| Command | Description |
|---------|-------------|
| [`/narutopromote`](leadership.md#narutopromote) | Promote players to special village ranks (Kage only) |
| [`/narutotransport`](leadership.md#narutotransport) | Manage transport locations and fast travel |

### Territory Commands

| Command | Aliases | Description |
|---------|---------|-------------|
| [`/territory`](territory.md#territory) | `/terr` | Create, manage, and claim territories |

### Admin Commands

| Command | Aliases | Description |
|---------|---------|-------------|
| [`/villagebank`](admin-commands.md#villagebank) | `/vbank`, `/vb` | Open and manage village banks |
| [`/villagebankpages`](admin-commands.md#villagebankpages) | `/vbankpages`, `/vbp` | Add pages to village banks |
| [`/villageranks`](admin-commands.md#villageranks) | | Manage village rank holders |
| [`/villagesreload`](admin-commands.md#villagesreload) | | Reload addon configurations |
