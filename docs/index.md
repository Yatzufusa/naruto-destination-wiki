## Quick Reference

| Symbol | Meaning |
|--------|---------|
| `<arg>` | Required argument |
| `[arg]` | Optional argument |
| `a\|b` | Choose one option |
| **OP** | Requires server operator |
| **Player** | Must be run by a player (not console) |

---

## Permission Levels

| Level | Who |
|-------|-----|
| **0** | All players |
| **2** | Server operators (OP) |

Some commands have additional checks beyond OP level, such as requiring a specific village rank or affiliation. These are noted per command.

!!! tip "The /destination Command"
    Use `/destination` as a master command hub. It auto-completes all sub-commands you have permission to use.
