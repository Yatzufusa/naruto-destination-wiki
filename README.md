# Naruto Destination - Wiki

Official wiki for **Naruto Destination**, a multiplayer-focused rework of the Mathioks Naruto Anime Mod for Minecraft 1.7.10.

**Live site:** https://yatzufusa.github.io/naruto-destination-wiki/

## What is Naruto Destination?

Naruto Destination overhauls the original Mathioks mod with scaling jutsus, a mastery system, a full token economy, and deep configurability — giving every server its own meta.

## Wiki Contents

- **Server Setup Guide** — Crucible server jar, recommended plugins and mods, modpack setup
- **Configuration Reference** — Every config file documented (stats, chakra, XP, DNA, enemies, tokens, resets, and more)
- **Command Reference** — All 38+ base mod commands with syntax, parameters, permissions, and examples
- **Addon: Villages** — Village banks, transport, territory, ranks, PvP, and diplomacy

- **Addon: XP Events** — Manual and automatic XP multiplier events
- **FAQ** — Singleplayer, multiplayer, progression, and game mechanics
- **Terms of Use** — Monetization rules and legal terms for servers
